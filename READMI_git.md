По твоему reflog серия коммитов надо возврать начинается с:
получения списки комитов 
git log --pretty=format:"%h  %ad  commit: %s" --date="format:%Y-%m-%d %H:%M:%S"

в такой формат
1515931  2026-03-09 08:06:40  commit: 2.03.2026 надо возврать
86deb16  2026-03-09 17:06:32  commit: 9.03.2026 надо возврать
2c47b30  2026-03-09 17:45:49  commit: 9.03.2026 надо возврать
7de0860  2026-03-10 03:16:47  commit: 9.03.2026 надо возврать
ce4f76f  2026-03-11 13:24:24  commit: 9.03.2026 надо возврать

Коммит перед этой серией — это:

606791a
Проверить
git show --no-patch --oneline 606791a
Вернуть main на него

Сейчас сделай так:

git switch main
git reset --hard 606791a
Проверить результат
git log --oneline -n 10

Если потом нужно, чтобы и удалённый origin/main стал таким же:

git push --force origin main

Но push --force делай только если точно хочешь переписать remote-историю.

Итог: твой целевой коммит — 606791a.
